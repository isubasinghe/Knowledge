# individuals\_and\_motivation

## Basic idea

How motivation, leadership, and team structure affect software-project outcomes. Covers Maslow, Herzberg, Tuckman, and the difference between management and leadership.

## Key facts

- Maslow: 5-level hierarchy of needs (physiological → safety → belonging → esteem → self-actualisation).
- Herzberg: hygiene factors *prevent dissatisfaction*; motivators *cause satisfaction* — they're separate.
- Tuckman: forming → storming → norming → performing → adjourning.
