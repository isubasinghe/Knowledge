# Project Schedule

## Basic idea

The project schedule answers "how long?" and "how much?". Standard notations: Gantt charts (bar-per-task on a calendar) and PERT charts (network of tasks with dependencies, slack, critical path).

## Key facts

- Gantt: best for communicating timeline + progress.
- PERT: best for analysing dependencies, slack, critical path.
- Both are typically generated from the same WBS.

## Project Schedule will answer two questions not answered so far

- How long will the system take to develop?
- How much will it cost?
