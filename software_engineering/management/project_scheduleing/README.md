# project\_scheduleing

## Basic idea

Breaking a project into tasks with dependencies, estimating durations, assigning resources, and tracking progress against the plan. Covers WBS, PERT/CPM networks, Gantt charts, milestones vs deliverables, and earned value tracking — for both plan-driven and agile contexts.

## Key formulas

- PERT three-point estimate: $e = (o + 4m + p)/6$
- Critical path: longest dependency chain — sets minimum project duration.
- Earned Value: $EV = \%\,\text{complete} \times BAC$; $SPI = EV/PV$; $CPI = EV/AC$.
- Schedule variance: $SV = EV - PV$; cost variance: $CV = EV - AC$.
