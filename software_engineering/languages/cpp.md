# C++

## Basic idea

C plus classes, templates, RAII, and operator overloading (Stroustrup, 1985). Multi-paradigm, zero-overhead-by-default, extraordinarily expressive — and extraordinarily complex.

## Key facts

- Year / creator: 1985, Bjarne Stroustrup at Bell Labs.
- Paradigm: OO + generic (templates) + procedural + functional.
- Typing: static, strong-ish.
- Memory: manual + RAII smart pointers (`unique_ptr`, `shared_ptr`).
- Standards: C++98/03, C++11, C++14, C++17, C++20, C++23.
