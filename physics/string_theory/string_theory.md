# String Theory

## Basic idea

Fundamental objects are 1D strings, not 0D points. Different vibrational modes = different particles; gravity emerges naturally from the closed-string spectrum.

## Key formulas

- Polyakov action: $S = -\dfrac{T}{2}\int d^2\sigma\,\sqrt{-h}\,h^{ab}\partial_a X^\mu \partial_b X_\mu$
- Nambu-Goto action: $S = -T \int d^2\sigma\, \sqrt{-\det(\partial_a X^\mu \partial_b X_\mu)}$
- String tension: $T = 1/(2\pi\alpha')$; string length $\ell_s = \sqrt{\alpha'}$
- Critical dimension: $D=26$ (bosonic), $D=10$ (superstring)
