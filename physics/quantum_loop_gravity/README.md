# quantum\_loop\_gravity

## Basic idea

A background-independent quantisation of general relativity. Space itself is granular at the Planck scale: area and volume have discrete spectra.

## Key formulas

- Wheeler-DeWitt equation: $\hat H \Psi = 0$
- Area operator eigenvalues: $A = 8\pi\gamma\ell_P^2 \sum_i \sqrt{j_i(j_i+1)}$ (Immirzi parameter $\gamma$)
- Planck length: $\ell_P = \sqrt{\hbar G/c^3}$
