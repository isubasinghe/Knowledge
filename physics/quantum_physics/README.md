# quantum\_physics

## Basic idea

The world at small scales: states are vectors in a Hilbert space, observables are Hermitian operators, and measurement outcomes are probabilistic.

## Key formulas

- Schrödinger equation: $i\hbar\partial_t\psi=\hat H\psi$
- Born rule: $P(x)=|\psi(x)|^2$
- Uncertainty: $\Delta x \Delta p \ge \hbar/2$
- Planck-Einstein: $E=\hbar\omega = hf$
